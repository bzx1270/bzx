# 小包写作工具

一个功能强大的小说写作工具，支持多种小说题材和自动更新功能。

## ✨ 功能特点

- **多种小说题材**：古代权谋、古代宫斗、古代搞笑、现代都市、玄幻修仙、科幻未来
- **智能AI生成**：基于AI模型生成高质量小说内容
- **Tokens管理**：支持微信充值和管理员手动调整
- **自动更新**：软件会自动检查并安装最新版本
- **历史记录**：保存写作历史，方便查阅
- **用户管理**：支持注册、登录和权限管理

## 🚀 快速开始

### 安装
1. 下载最新版本的软件
2. 解压到任意目录
3. 运行 `xiaobao_write.exe`（如果没有EXE文件，运行 `xiaobao_write.py`）

### 登录
- **管理员账号**：admin / admin123
- **普通用户**：注册即可获得10万Tokens

### 充值
1. 点击 "💎 微信充值" 按钮
2. 选择充值套餐
3. 扫码支付并填写备注（订单号）
4. 点击 "✅ 手动确认到账"
